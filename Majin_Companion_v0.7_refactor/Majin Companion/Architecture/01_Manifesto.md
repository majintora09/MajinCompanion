# 01 - Manifesto

## The Companion Oath

Majin Companion will never try to replace Yuri.
It exists to help yesterday's Yuri leave enough behind that today's Yuri can continue, and tomorrow's Yuri can go further.

## Mission

Reduce the friction between yesterday's Yuri and today's Yuri.

## Definitions

- Dream: an idea that is not actionable yet.
- Project / Place: a dream Yuri decided deserves to exist.
- Session: a snapshot of Yuri's brain while making a project real.
- Workshop: the place where projects live.
- Brain: the memory of one place.
- Future Yuri: the version of Yuri that never forgets.

## Non-negotiables

- Memory over reminders.
- Context over data.
- Continue over start.
- Companion suggests, Yuri decides.
- Never punish. Only encourage.
- Never interrupt flow unless the value is obvious.
- Reduce friction.
- The AI remembers. The human creates.
